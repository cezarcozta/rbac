### Getting Started  

>**Pré-conditions:**
>
> - **yarn**  
> - **npm**  

- clone this repository  
`git clone https://github.com/cezarcozta/rbac.git`  

- access the repository folder  
`cd rbac`  

- install all dependencies  
`yarn`  

- run all migrations  
`yarn typeorm migration:run`  

- run the server  
`yarn start`
